# Review Questions

**1. What is the Internet?**
The Internet is a vast, global network of interconnected computers and networks that communicate using standardized protocols (such as TCP/IP). It serves as the underlying physical infrastructure that connects billions of devices worldwide, enabling them to exchange information.

**2. Explain the difference between the Internet and the World Wide Web.**
The Internet is the global network of interconnected hardware and cables that allows computers to communicate. The World Wide Web (the Web), on the other hand, is a service that runs *on top of* the Internet. The Web consists of interconnected documents (web pages) and multimedia resources accessed via web browsers using the HTTP/HTTPS protocols. 

**3. What is the role of an ISP?**
An ISP (Internet Service Provider) is a company that provides individuals and organizations with access to the Internet. ISPs supply the necessary infrastructure, routing, and equipment to connect users' devices to the broader global network.

**4. What is an IP address?**
An IP (Internet Protocol) address is a unique numerical label (or alphanumeric in IPv6) assigned to every device connected to a computer network. It serves two primary functions: identifying the specific device or host, and indicating its location on the network so data can be correctly routed to it.

**5. What does DNS do?**
DNS (Domain Name System) acts as the "phonebook" of the Internet. It translates human-readable domain names (like `google.com`) into machine-readable IP addresses (like `142.250.190.46`), allowing browsers to locate and load websites without requiring users to memorize complex strings of numbers.

**6. Explain the difference between HTTP and HTTPS.**
HTTP (Hypertext Transfer Protocol) is the standard protocol for transmitting data across the web. HTTPS (Hypertext Transfer Protocol Secure) is the secure version of HTTP. In HTTPS, the communication between your browser and the website is encrypted using SSL/TLS, protecting sensitive data (like passwords and credit card numbers) from being intercepted.

**7. Mention three benefits of the Internet.**
- **Instant Communication:** Enables seamless global communication through email, video conferencing, and instant messaging.
- **Access to Information & Education:** Provides immediate access to a vast, limitless repository of knowledge, research, and learning materials.
- **E-commerce & Convenience:** Allows users to shop, bank, work, and access services remotely from anywhere in the world.

**8. List three Internet security best practices.**
- **Use Strong Passwords & MFA:** Create strong, unique passwords for different accounts and enable Multi-Factor Authentication (MFA) wherever possible.
- **Keep Software Updated:** Regularly update operating systems, web browsers, and antivirus software to patch security vulnerabilities.
- **Beware of Phishing:** Avoid clicking on suspicious links, downloading unverified attachments, or sharing sensitive information with unknown sources.
